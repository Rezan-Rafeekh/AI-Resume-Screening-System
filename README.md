# AI-Resume-Screening-System
Companies receive thousands of resumes. This project uses Machine Learning + NLP to:  Automatically screen resumes,  Match candidates to job descriptions,  Give a job recommendation score

## Features
- Resume and job description matching
- Machine Learning using TF-IDF
- Cosine similarity score
- Simple Streamlit web interface

## Technologies Used
- Python
- Machine Learning
- NLP
- Scikit-learn
- Streamlit

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   streamlit run app.py

## Use Case
This system helps recruiters automatically screen resumes and find suitable candidates efficiently.
